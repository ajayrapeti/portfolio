- [ ] Run `npm run dev` and check for errors
- [ ] Test smooth scrolling between sections

- [ ] Share your portfolio link!

## 📝 Quick Commands

```bash

npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
